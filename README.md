# Sales Data Analysis Project

In this project, I explored a retail sales dataset using Python to better understand what affects profitability.

During the analysis, I worked with 9,994 transactions and focused on identifying patterns in profit, discounts, and product performance.

## What I found

One of the key findings was that not all sales are actually profitable. Around 19% of transactions result in losses, which is quite significant.

I also discovered a clear relationship between discounts and profit. As discounts increase, profitability decreases. In fact, transactions with discounts above 30% are usually unprofitable, with the biggest losses happening at 50%.

Another interesting insight is related to product categories. While some products like copiers and phones generate strong profits, others perform poorly. In particular, tables turned out to be the biggest source of losses.
